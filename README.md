# 📊 Análise de Dados de Varejo: Do SQL ao Power BI

Este repositório contém um projeto prático desenvolvido para consolidar conceitos de Banco de Dados Relacional e Visualização de Dados, unindo os conhecimentos acadêmicos da faculdade de Engenharia de Software com ferramentas de mercado.

## 💻 Sobre o Projeto
O objetivo do projeto foi extrair dados brutos de planilhas eletrônicas de uma empresa de varejo fictícia, estruturá-los em um banco de dados relacional e criar uma narrativa visual (storytelling) para apoiar a tomada de decisão gerencial.

## 🛠️ Etapas Técnicas Realizadas
* Ingestão & Modelagem: Importação dos dados brutos e estruturação no banco de dados SQL, aplicando regras de integridade referencial (Chaves Primárias e Estrangeiras).
* Manipulação e Limpeza: Criação de queries SQL para tratar, filtrar e garantir a consistência das informações de vendas.
* Visualização: Conexão do banco de dados ao Power BI para a construção de dashboards dinâmicos.

## 📌 Origem dos Dados
A base de dados bruta utilizada faz parte de um estudo de caso prático de um curso de 2 horas realizado na plataforma Udemy. Toda a implementação dos scripts SQL e a interface do dashboard foram desenvolvidas por mim como forma de aprendizado prático.

## 🚀 Tecnologias Utilizadas
* SQL / Sistema de Gerenciamento de Banco de Dados (SGBD)
* Power BI
